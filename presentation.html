חוויית האורח — 30 שניות מסריקה לבקשה

📱
  1. אורח מתיישב בלובי, רואה פוסטר עם QR
  2. פותח מצלמה, סורק — נכנס מיד ל-PWA
  3. רואה: "ברוכים הבאים למלון דן תל אביב" + לוגו
  4. בוחר שפה (HE / EN / AR / RU)
  5. ממלא שם וחדר → לוחץ "כניסה"
  6. שולח בקשה: "מגבת נוספת", "תקלה במזגן", "בקשת WAKE-UP"

Multi-tenant — מיועד גם לרשתות

למלון בודד

  • אשף הקמה ב-3 שלבים
  • הזמנת עובדים בלחיצה
  • לוגו + צבעי מותג
  • חדרים: בולק-אד אוטומטי

לרשת מלונות

  • סופר אדמין שמנהל את כולם
  • הפרדה מלאה (Tenant Isolation)
  • נתונים בין מלונות לעולם לא דולפים
  • השוואת KPIs בין סניפים

הסטאק הטכני

Frontend

  • Flutter — אפליקציית עובדים (iOS / Android / Web)
  • Flutter Web — PWA לאורחים
  • Next.js 15 — פאנל אדמין
  • Tailwind — עיצוב

Backend

  • Supabase — PostgreSQL + Auth + Storage
  • Realtime — WebSocket עדכונים חיים
  • Edge Functions — Deno (push, invite, export)
  • OneSignal — Push Notifications

☁️ Vercel (Admin) · Netlify (PWA) · Supabase Cloud · GitHub Actions CI

אבטחה ופרטיות

  • ✓ RLS (Row-Level Security) על כל טבלה — מלון לא רואה נתונים של מלון אחר
  • ✓ JWT עם הרשאות מבוססות תפקיד
  • ✓ MFA / TOTP לסופר אדמין
  • ✓ סשן אוטומטי ננעל אחרי X דקות אי-פעילות
  • ✓ אורח אנונימי לא רואה דבר מלבד שם המלון + לוגו
  • ✓ תמונות ב-Private Bucket — נגיש רק עם token חתום
  • ✓ אודיט-לוג מלא של פעולות מנהל
