האורח שלכם מצפה לרמת שירות גבוהה — אבל הכלים שלכם מהמאה הקודמת.

📞
"מצטערים, האחזקה תגיע בעוד שעתיים"
בקשה מאורח = שיחת טלפון לקבלה → סמס למחלקה → פוסטיט. אורח מבוגר לא רוצה להתקשר 3 פעמים על מגבת.
🌐
תיירות אילת — מגוון תרבותי
אורחים מאירופה, רוסיה, וצפון אמריקה. קבלה דוברת עברית/אנגלית בלבד — חיכוך באורחים דוברי ערבית ורוסית.
📊
חוסר נראות לניהול
מנהל המלון לא יודע בזמן אמת: כמה בקשות פתוחות? אילו מחלקות עמוסות? אילו אורחים פחות מרוצים מהשירות?
⭐
דירוג TripAdvisor / Booking לא קופץ למעלה
אורח לא מרוצה רושם בסקירה ציבורית. בלי משוב מובנה במהלך השהות — מאחרים בכל פעם.

חוויית האורח — 30 שניות מסריקה לבקשה

📱
  1. אורח בוגר מגיע ללובי, רואה פוסטר אלגנטי עם QR
  2. פותח מצלמה, סורק — נכנס מיד ל-PWA
  3. רואה: "ברוכים הבאים למלון רוקסון רד סי אילת" + לוגו המלון
  4. בוחר שפה (עברית · English · العربية · Русский)
  5. ממלא שם וחדר → לוחץ "כניסה"
  6. שולח בקשה: "מגבת נוספת", "WAKE-UP בשעה 7:30", "הזמנת שולחן בחדר אוכל"

✓ ללא Download. ללא הרשמה. ללא סיסמה. נגיש לכל גיל.

למה זה חשוב לרוקסון?

  • אילת = תיירות נכנסת מאירופה (אנגלית, רוסית)
  • אורחים ערבים מירדן ומהאזור
  • טיסות צ'רטר מרוסיה ואוקראינה (15% מהאורחים)
  • צוות מגוון — עברית/ערבית/רוסית

איך זה עובד?

  • RTL / LTR אוטומטי
  • שמירת בחירת שפה למכשיר
  • תרגום מלא — לא Google Translate
  • הוספת שפה נוספת = ~יום עבודה

Multi-tenant — מיועד גם לרשתות

למלון בודד

  • אשף הקמה ב-3 שלבים
  • הזמנת עובדים בלחיצה
  • לוגו + צבעי מותג
  • חדרים: בולק-אד אוטומטי

לרשת מלונות

  • סופר אדמין שמנהל את כולם
  • הפרדה מלאה (Tenant Isolation)
  • נתונים בין מלונות לעולם לא דולפים
  • השוואת KPIs בין סניפים

הסטאק הטכני

Frontend

  • Flutter — אפליקציית עובדים (iOS / Android / Web)
  • Flutter Web — PWA לאורחים
  • Next.js 15 — פאנל אדמין
  • Tailwind — עיצוב

Backend

  • Supabase — PostgreSQL + Auth + Storage
  • Realtime — WebSocket עדכונים חיים
  • Edge Functions — Deno (push, invite, export)
  • OneSignal — Push Notifications

☁️ Vercel (Admin) · Netlify (PWA) · Supabase Cloud · GitHub Actions CI

אבטחה ופרטיות

  • ✓ RLS (Row-Level Security) על כל טבלה — מלון לא רואה נתונים של מלון אחר
  • ✓ JWT עם הרשאות מבוססות תפקיד
  • ✓ MFA / TOTP לסופר אדמין
  • ✓ סשן אוטומטי ננעל אחרי X דקות אי-פעילות
  • ✓ אורח אנונימי לא רואה דבר מלבד שם המלון + לוגו
  • ✓ תמונות ב-Private Bucket — נגיש רק עם token חתום
  • ✓ אודיט-לוג מלא של פעולות מנהל
